# ActionMultiRPG_Unreal

3인칭 액션 멀티플레이 RPG 게임 (Unreal Engine 5 기반)

멀티 보스 레이드, 솔로잉 던전, 전투 시스템(스킬, 회피), 인벤토리 및 장비 시스템 등 다양한 콘텐츠를 포함한 클라이언트 프로젝트입니다.

---

## 주요 기능

- 실시간 전투 시스템 (스킬, 회피)
- 멀티플레이 보스 레이드
- 싱글 던전 콘텐츠
- 인벤토리, 아이템, 장비 장착 시스템
- C++ 서버와의 연동 (플레이어 상태 및 전투 정보 통신)

---

## 개발 환경

- Unreal Engine 5.3
- 언어: C++, Blueprint 혼합
- 플랫폼: Windows
- 버전 관리: Git + GitHub
